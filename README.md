## Installation Local Project

```elixir 
git clone https://github.com/rysxa/mii-react.git
```

then

```elixir 
npm install
```

- Config *.env* like this

```elixir
NODE_DEV=development
```

## Run Local Project

```elixir 
npm run dev
```
